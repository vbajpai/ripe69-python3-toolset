Installation and Usage:
-----------------------

Make sure `python3` and `pip3` are installed on your machine.

    % which python3
    % which pip3

Install Dependencies:

    % make

Run the IPython Notebook:

    % make run

View the IPython Notebook at [0.0.0.0:7777](http://0.0.0.0:7777)
